CAP-01-06 — Avaliação Inicial e Anamnese

Identificação

ID: CAP-01-06

Título: Avaliação Inicial e Anamnese

Vision: VISION-01 — Registro da Prática Clínica

Status: draft

Descrição

Capacidade de registrar a história profunda e multidimensional do paciente no início do tratamento. Esta funcionalidade captura a queixa principal, antecedentes familiares, marcos de desenvolvimento e o exame das funções mentais, servindo como a base de dados estrutural para todo o prontuário.

Objetivos

Mapeamento Biopsicossocial: Estruturar a história de vida do sujeito em dimensões (pessoal, familiar, social).

Registro de Queixa: Documentar o motivo inicial da busca por terapia em formato narrativo.

Exame de Funções Mentais: Registrar o estado basal do paciente (atenção, humor, cognição).

Referência Rápida: Facilitar a consulta a dados históricos que não mudam a cada sessão.

Contexto Clínico (SOTA)

A Anamnese é a fundação. No padrão Arandu, ela utiliza o "Progressive Disclosure", permitindo que o terapeuta preencha os dados ao longo dos primeiros encontros, mantendo a imersão na escuta através do "Silent Input".

Requisitos Relacionados

REQ-01-06-01: Anamnese Clínica Multidimensional.

Valor de Negócio / Clínico

Centraliza informações vitais que muitas vezes ficam perdidas em cadernos ou memórias fragmentadas. É essencial para a segurança do paciente e para a continuidade do cuidado em casos de interconsulta.

Critérios de Sucesso

O sistema permite salvar seções da anamnese de forma independente via HTMX.

O conteúdo clínico utiliza obrigatoriamente a fonte Source Serif 4.

Os dados estão isolados no banco de dados específico do profissional.
