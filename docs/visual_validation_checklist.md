# Checklist de Validação Visual

## Desktop (1920x1080)
- [ ] Top Bar não cobre conteúdo
- [ ] Sidebar não sobrepõe main canvas
- [ ] Cards não estouram largura
- [ ] Scroll vertical funciona sem cortes

## Mobile (375x667)
- [ ] Sidebar vira drawer (hambúrguer)
- [ ] Bottom Nav aparece se necessário
- [ ] Textos não vazam horizontalmente
- [ ] Botões são clicáveis (zona do polegar)

## Tablet (768x1024)
- [ ] Layout se adapta entre mobile/desktop
- [ ] Sidebar comportamento correto
- [ ] Grid 65/35 se mantém legível

## Comum
- [ ] Nenhum elemento com position:fixed sem z-index
- [ ] Cores do Design System respeitadas
- [ ] Tipografia clínica (Source Serif) em conteúdo
